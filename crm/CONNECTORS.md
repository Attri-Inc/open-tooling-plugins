# Connectors

The CRM plugin works standalone with Open Tooling CRM's MCP tools. Connect these for supercharged workflows:

| Category | Tool | What It Powers |
|----------|------|----------------|
| Email | Gmail, Outlook | Auto-ingest emails as artifacts, extract observations |
| Calendar | Google Calendar, Outlook Calendar | Log interactions, prep for meetings |
| Chat | Slack, Teams | Capture conversation context as artifacts |
| Documents | Google Drive, OneDrive, Notion | Ingest documents and extract structured data |
| Enrichment | Clay, ZoomInfo, Apollo | Auto-populate contact and company data |

None of these are required. The plugin works fully with just the Open Tooling CRM MCP server connected.
